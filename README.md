# Sumer Registration System

A Node.js web application that demonstrates server-side form submission using Express.

## Features
- User registration form
- Server-side form handling
- Dynamic response page
- Clean UI design

## Technologies Used
- Node.js
- Express.js
- HTML
- CSS

## Run the Project

1. Install dependencies
npm install

2. Start server
node server.js

3. Open browser
http://localhost:3000/sumerproject
